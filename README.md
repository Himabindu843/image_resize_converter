
# Image Resizing and Format Converter

This Python project resizes images and converts them between JPG, PNG, and BMP formats.

## Features
- Upload image in Google Colab
- Resize image to given dimensions
- Convert image formats (JPG, PNG, BMP)

## Requirements
- Python 3
- Pillow library

## How to Run
1. Upload the image
2. Update the filename in code
3. Run the script
4. Download the output images
